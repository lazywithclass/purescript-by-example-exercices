purescript-by-example-exercices
===============================

Starter project for PureScript following https://leanpub.com/purescript/read

## Install

### Prerequisites

* Ubuntu Linux
* nodejs
* ~/bin in `$PATH`
* ~/.cabal/bin in `PATH`

```sh
$ make install
```

## Project setup

```sh
$ mkdir /path/to/your-purescript-project
$ cd /path/to/your-purescript-project
$ grunt-init purescript

$ npm install
$ bower update
```

## Building 

```sh
$ grunt
```
